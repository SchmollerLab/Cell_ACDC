# Cell-ACDC keyboard shortcuts

## Core edit tools (customizable)

These five tools are listed in **Settings → Customize keyboard shortcuts…** (`Ctrl+K`).

| Key | Tool | Usage |
|-----|------|--------|
| `B` | Brush | Left-click / drag to paint (double-`B` = power brush) |
| `E` | Eraser | Left-click / drag to erase (double-`E` = power eraser) |
| `P` | Curvature tool | Left-click to place spline anchors |
| `I` | Edit ID | Left-click a cell to change its ID |
| `X` | Delete ID | Left-click a cell to delete it |

All toolbar tools use **activate tool → left-click** on the image.

---

## Navigation & file (not in shortcut editor)

| Key | Action |
|-----|--------|
| `←` / `→` | Previous / next frame |
| `H` | Zoom to cells; double-`H` = reset zoom |
| `Ctrl+S` | Quick save |
| `Ctrl+Z` / `Ctrl+Y` | Undo / redo |
| `Ctrl+F` | Find ID |
| `Esc` | Cancel / clear highlights |

See the GUI menus for additional bindings (slideshow, auto-pilot, cell-cycle table, etc.).

---

*Custom bindings are saved in `shortcuts.ini` and override these defaults.*
